# Vita App

## Descripción del proyecto
VitaApp es una aplicación móvil enfocada en la gestión y análisis de la ingesta nutricional diaria. Permite a los usuarios registrar los alimentos consumidos, consultar el total de calorías diarias, visualizar nutrientes principales y controlar sus entrenamientos.  
La app ayuda a cumplir metas fitness como pérdida de peso, mantenimiento o aumento de masa muscular, ofreciendo límites personalizados y un historial de registros diarios para el seguimiento del progreso.  
Actualmente el proyecto se encuentra en **etapa de prototipo**.

## Tecnologías utilizadas
- **Kotlin**
- **Android Studio**
- **Jetpack Compose**
- **Material 3**
- **Gradle (Kotlin DSL)**

## Instrucciones para ejecutar la app
1. Clona el repositorio.
2. Abre el proyecto en **Android Studio**.
3. Espera a que Gradle sincronice dependencias.
4. Conecta un dispositivo Android o inicia un emulador.
5. Ejecuta la app desde Android Studio con el botón **Run**.
