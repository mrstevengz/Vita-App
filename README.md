## Descripción del Proyecto

VitaApp es una aplicación móvil enfocada en la gestión y análisis de la ingesta nutricional diaria. Su objetivo principal es permitir que los usuarios lleven un control detallado de sus calorías, nutrientes, comidas y entrenamientos diarios. Está diseñada para ayudar a las personas a cumplir sus metas fitness, ya sea bajar de peso, mantenerlo o ganar masa muscular, mediante el establecimiento de límites personalizados.

La aplicación funciona como un tracker inteligente donde el usuario puede registrar los alimentos consumidos durante el día, consultar el total de calorías ingeridas, revisar macronutrientes clave (proteínas, carbohidratos y grasas) y llevar un seguimiento visual de su progreso. Con esta solución, buscamos reducir el tedio del control manual de comidas y ofrecer una herramienta práctica que fomente la disciplina y mejore los hábitos nutricionales.

*Nota: El proyecto se encuentra actualmente en fase de prototipado, por lo que algunas funcionalidades y accesos desde el dashboard principal están en desarrollo activo.*

## Tecnologías Utilizadas

El desarrollo de la aplicación se está llevando a cabo con herramientas modernas orientadas a la creación de aplicaciones nativas escalables y con buenas prácticas de arquitectura:

* **Kotlin:** Lenguaje de programación principal.
* **Android Studio:** Entorno de desarrollo integrado (IDE).
* **Jetpack Compose:** Toolkit moderno para la creación de interfaces de usuario (UI) nativas de forma declarativa.
* **Material 3:** Sistema de diseño para una experiencia de usuario accesible y visualmente coherente.
* **Gradle (Kotlin DSL):** Sistema de automatización de construcción y gestión de dependencias.

## Instrucciones para Ejecutar la App

Como el proyecto está en etapa de desarrollo, sigue estos pasos para probar el prototipo en tu entorno local:

1. **Clonar el repositorio:**
   Abre tu terminal y ejecuta el siguiente comando:
   ```bash
   git clone https://github.com/mrstevengz/Vita-App.git
   ```

2. **Abrir el proyecto:**
   * Inicia **Android Studio**.
   * Selecciona `File > Open` y busca la carpeta donde clonaste el repositorio.

3. **Sincronizar Gradle:**
   * Espera a que Android Studio descargue las dependencias necesarias y sincronice los archivos de Gradle.

4. **Ejecutar la aplicación:**
   * Conecta un dispositivo Android físico mediante depuración USB o inicia un Emulador de Android (AVD).
   * Haz clic en el botón **Run** (el ícono de reproducción verde) en la barra de herramientas superior, o usa el atajo `Shift + F10`.
